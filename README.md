Steps to run the project
Android
1. Download Expo Go App from the Play Store
2. Run `npx expo start`
3. Open the Expo Go App and scan the QR code displayed in terminal

Type of Error Handling
1. Network
2. Authentication
3. Data
4. Navigation
5. Server side
